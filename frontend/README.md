# Laundry Management System

Aplikasi management laundry berbasis Flutter dan Golang.

## Teknologi

### Frontend
- Flutter
- Provider State Management
- HTTP REST API
- Shared Preferences

### Backend
- Golang
- Gin Framework
- GORM ORM
- JWT Authentication
- bcrypt Password Hashing

### Database
- MySQL
